# Kruskal Algorithm

간선 중심으로 최소 신장 트리를 구하는 알고리즘으로, 간선이 적을 때 유리합니다.<br>
```
💡 크루스칼 알고리즘 구현 방법
1. 간선들을 가중치의 오름차순으로 정렬합니다.
2. 가중치가 가장 작은 간선을 선택하여 그 간선이 지금까지 만들어진 MST와 사이클을 형성한다면 제외하고, 아니면 MST에 추가합니다.
3. 모든 간선에 대해 반복합니다.
```
